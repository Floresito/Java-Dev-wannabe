En mi caso ya se tenia instalado JDK 17, se documentará el proceso de instalación del JDK 24
Se utilizará el JDK 24 junto con Apache NetBeans como IDE

Para instalar JDK 24:

Ya que se utilizará Windows, del siguiente link obtenemos el x64 installer:

https://www.oracle.com/europe/java/technologies/downloads/#jdk24-windows



Solamente se debe seguir el wizard de instalación durante este proceso.

En mi caso instalaré el SDK en la siguiente ruta:

C:\\Program Files\\Java\\jdk-24\\



Una vez seleccionada la ruta clic en "Next" para comenzar la instalación.

Finalizada la instalación del SDK procedemos con la instalación del IDE



El enlace para descargar el instalador es el siguiente:

https://netbeans.apache.org/front/main/download/nb25



De la sección "Installers and Packages" descargamos el que tenga por nombre:

"Apache-NetBeans-25-bin-windows-x64.exe"



Seguimos el wizard de instalación, en este caso instalaremos todos los paquetes

por default, que son los siguientes:

* Base IDE
* Java SE
* Java EE
* HTML5/JavaScript
* PHP



Después de aceptar el acuerdo de licencia podremos seleccionar la ruta de

instalación para Apache NetBeans, en este caso dejaremos la ruta por default:

C:\\Program Files\\Java\\NetBeans-25



En el caso de que se tenga otra version del SDK instalada y se desee usarla

se debe cambiar la ruta de "JDK™ for the Apache NetBeans IDE" a la ruta donde se tenga instalada dicha versión del SDK



Después de confirmar los datos se procede con la instalación
